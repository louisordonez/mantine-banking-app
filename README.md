This project is a combination of Banking and Budget web application.

## Tech Stack

- HTML
- SCSS
- React
- Mantine

## Features

### Admin:

- Create, edit, and remove a user
- View all user transactions

## User:

- Create an account
- Withdraw, deposit, and transfer money
- Add, edit, and remove expenses
- View transactions
